# 3D&T Victory DLC Project

Bem-vindo ao repositório 3D&T Victory DLC! Este espaço reúne livros completos, suplementos e materiais de referência para 3DeT Victory", criado para o sistema de RPG de mesa originalmente desenvolvido por Marcelo Cassaro e publicado pela Jambô Editora.

Além do material atual, o repositório está preparado para receber novos livros DLC, expansões e conteúdos feitos por fãs para 3D&T Victory. Contribuidores são incentivados a propor e desenvolver cenários, suplementos ou livros de regras adicionais, expandindo o universo de 3D&T Victory com ideias e aventuras inéditas.

## Estrutura do Repositório

- **Academia Arcana Victory/Livro/**: Main book chapters and character sheets.
- **Academia Arcana Victory/Suplementos/**: Campaign supplements, additional rules, and narrative expansions.

## Como Contribuir

Quer ajudar a desenvolver materiais para este projeto de TTRPG? Siga estas orientações:

- **Crie a pasta do seu DLC** seguindo a estrutura de pastas já existente.
- **Crie novos capítulos ou suplementos** seguindo a estrutura e convenções de nomes já existentes.
- **Documente mudanças importantes** no arquivo de plano de revisão (`referencias/PLANO_REVISAO.md`).
- **Inclua sempre os créditos obrigatórios** em qualquer novo material criado.
- **Não publique conteúdo comercialmente** nem utilize artes oficiais sem permissão.
- **Deixe claro** que todo conteúdo é não-oficial e feito por fãs.

### Sugestão de Mudanças

- Para sugerir alterações, **abra uma Pull Request (PR)**:
    1. Fork este repositório.
    2. Crie uma branch com suas modificações.
    3. Descreva claramente suas mudanças no título e na descrição do PR.
    4. Aguarde revisão e feedback dos mantenedores.

### Dúvidas, Sugestões ou Colaborações

- Para dúvidas ou sugestões, **abra uma Issue** explicando sua ideia ou problema.
- Você também pode **contatar o mantenedor** diretamente pelo GitHub.

Colabore, compartilhe ideias e ajude a fortalecer o RPG brasileiro!
