## Estrutura Temática

Este livro é sobre magia que vem de dentro. As quatro franquias cobertas partilham um axioma: o poder não existe fora da pessoa. A varinha de transformação não é fonte — é chave. O poder estava lá antes da chave existir. A questão que todas estas histórias fazem, de formas distintas, é: *quando você descobre quem é, o que faz com isso?*

### O Arco de Quatro Registros

| Capítulo | O que a magia é | A questão central | Tom |
|----------|----------------|-------------------|-----|
| Sentinelas do Solstício | Identidade encarnada | Quem sou quando estou completa? | Épico |
| Guardiãs de Coleção | Cuidado como prática mágica | O que proteço quando proteço? | Suave, quente |
| Parceiras de Ressonância | Convicção materializada em força | Posso recusar aceitar que é o fim? | Físico, direto |
| Cavaleiras do Vórtice | Vontade que move o mundo | Sirvo uma causa que escolhi? | Maduro, pesado |

### Como Usar Este Livro

**Para uma campanha de identidade:** Use o cap. 01 (Sentinelas do Solstício) como ponto de entrada. O sistema Despertar Planetário serve como motor de progressão para qualquer campanha onde os PJs descobrem quem realmente são ao longo do tempo.

**Para uma campanha modular por sessões:** Use o cap. 02 (Guardiãs de Coleção). Cada Artefato Arcanamente Independente é uma sessão auto-contida; o arco de fundo conecta as sessões sem exigi-las na ordem.

**Para uma campanha de ensemble ativo:** Use o cap. 03 (Parceiras de Ressonância) com o Subsistema Poder de Amizade. Máximo impacto quando todos os PJs adotam o subsistema.

**Para um arco de clímax dramático:** Use o cap. 04 (Cavaleiras do Vórtice) como a revelação de que o sistema que os PJs serviram tem custo que ninguém contou completamente. O Sistema de Pilar é o melhor motor para arcos de fundação narrativa.

## Mecânica Unificada de Transformação

A mecânica central do livro está no Cap. 06, mas a versão resumida: toda personagem com poderes de identidade pode ter 1 **Estado Transformado** que custa 1 PM para ativar, dura toda uma cena, e concede Ganho em testes do domínio central do personagem. Os subsistemas específicos de cada franquia se sobrepõem a essa base sem contradizê-la.
