# Livro 3 — Palavras de Poder: O Livro dos Nomes

> *"Há uma diferença entre saber o que uma coisa faz e saber o que uma coisa é. A primeira é técnica. A segunda é poder."*
> — Bibliotecário do Terceiro Andar

---

## Identidade do Livro

**Tom:** Contemplativo, lírico, rigoroso  
**Premissa:** Entender é poder. Nomear é comandar. Compreender é arriscar.  
**Pergunta central:** *O que significa realmente conhecer algo — e o que você está disposto a pagar por esse conhecimento?*

## Regra de Ouro Mecânica

> **Este livro adiciona no máximo 2 subsistemas simultâneos à sua campanha.**
>
> Escolha UM sistema de magia-por-compreensão (Simpatia/Alar de Kingkiller **ou** Língua dos Nomes de Eragon **ou** Nomes Verdadeiros de Y3) como base. Use T4 como guia de calibração, não como subsistema adicional. A aventura funciona com qualquer dos três.

## Notas Editoriais

- **Unificação Kingkiller + Eragon + Y3:** Os três suplementos usam a premissa "compreensão = poder" mas por ângulos diferentes. O Cap. 00 estabelece o framework unificado; os caps. 01-03 são instâncias. O GM não precisa usar os três — qualquer um funciona sozinho.
- **T4 como meta-ferramenta:** O Cap. 04 (Hard/Soft) não é um subsistema de jogo — é uma ferramenta de design para o Mestre calibrar qualquer sistema mágico. Funciona como "manual do construtor" independente do tema.
- **Aventura agnóstica:** "O Nome Que o Grimório Esqueceu" foi escrita para funcionar com qualquer dos três sistemas de nomeação — ou sem nenhum, usando apenas mecânica base de 3D&T Victory.
