# Livro 4 — Sangue & Legado: O Livro da Herança

> *"Ninguém escolhe onde nasce. Mas todo mundo escolhe o que faz com o que herdou — e esse momento, o momento da escolha, é quando a herança deixa de ser corrente e vira fundação."*
> — Professora Wynna Valdris, cerimônia de admissão dos grêmios

---

## Identidade do Livro

**Tom:** Aventureiro, dramático, institucional
**Mecânica Central:** Herança como fonte de poder e como peso narrativo
**Premissa:** O que você herdou define quem você é — ou é só onde você começa?
**Pergunta central:** *O que você merece por nascer vs. o que conquista por escolha?*

---

## Para Quem É Este Livro

Para mesas que querem jogar com personagens definidos pelo que veio antes deles. Linhagens mágicas, heranças divinas, sangue angelical, famílias com história, instituições com séculos de expectativas. O poder aqui não é descoberto — é *herdado*. E o drama vem de decidir o que fazer com uma herança que ninguém pediu.

Se Estrelas & Cristais é sobre descobrir quem você é, e Tinta & Sombras é sobre sobreviver ao que a escola exige, **Sangue & Legado** é sobre **carregar o peso do que veio antes** — e escolher se esse peso é proteção ou prisão.

Este é o livro com o maior volume de opções de personagem da linha. Três origens de linhagem com kits, vantagens e subsistemas próprios. Onze arquétipos acadêmicos. Cinco sociedades secretas. Três kits de escola visitante. Uma campanha de intercâmbio de 8–12 sessões e um arco de torneio de 3 sessões.

**Use a Tabela Comparativa de Opções de Personagem (Apêndice) para navegar.** Ela existe para isso.

## Estrutura Temática

Este livro é sobre magia que está no sangue. As três franquias cobertas partilham um axioma: o poder antecede o indivíduo. Você não descobre seus poderes — eles estiveram esperando desde antes de você nascer. A varinha escolhe o bruxo. O sangue divino acorda no momento certo. Os glifos reagem ao sangue angelical que você não pediu para ter.

### Os Três Paradigmas de Herança

| Franquia | O que é a herança | O que define pertencimento | O custo da herança | Tom |
|----------|-------------------|---------------------------|--------------------|----|
| Academia Thornwall | O Foco revela identidade + Casas de caráter | Casas por revelação do Livro das Casas | Expectativas, nome familiar, destino institucional | Aventureiro, escolar |
| Campo das Convergências | Sangue arcano-ancestral da Era das Arcas | Células por divindade progenitora | Entidades rastreiam, origens ausentes, destino não pedido | Heroico, mitológico |
| Ordem dos Glifeiros | Sangue arcano-ancestral + inscrição ritual | Hierarquia institucional | Dor dos glifos, peso da Ordem, hierarquia opressora | Sombrio, institucional, político |

### O Que as Três Têm em Comum

Todas perguntam a mesma coisa: **o que você faz com o que não escolheu?**

Eliot não escolheu a Casa Âncora — o Livro das Casas revelou. Kael não escolheu ser descendente do Senhor das Marés. Os Glifeiros não escolheram ter sangue arcano-ancestral. A Academia Victory não escolhe alunos por linhagem — mas muitos deles chegam com uma.

O framework genérico de Linhagem (cap00) transforma essa pergunta em mecânica: **Traço de Linhagem** (o que você herdou), **Fardo de Linhagem** (o que isso custa), e **Status de Herança** (como o mundo responde à sua origem). As três franquias são instâncias diferentes dessa mesma estrutura.

---

## Regra de Ouro

> **Este livro oferece três paradigmas de herança. Escolha UM como base da campanha.**
>
> Academia Thornwall funciona como base quando a campanha é sobre escola, casas, pertencimento e rivalidade entre grêmios. Campo das Convergências funciona quando o poder vem de sangue literal e a ameaça é externa. Ordem dos Glifeiros funciona quando a instituição é o cenário E o antagonista.
>
> Você pode usar os três como referência e inspiração. Mas não tente rodar Thornwall + Campo + Glifeiros como um único worldbuilding coerente. Cada paradigma tem cosmologia própria. Misturar cria sopa, não cenário.
>
> O framework genérico de Linhagem (cap00) existe para quem quer o *tema* de herança sem se comprometer com nenhum paradigma específico. Use-o e construa a partir dele.

## Como Usar Este Livro

**Para uma campanha de escola clássica com herança como tema:**
Use cap00 + cap01 (Academia Thornwall) como base. O mapeamento de Casas→Grêmios é a ponte mais direta para uma campanha de AAV onde linhagem importa. Adicione cap04 (Sociedades Secretas) para camadas de intriga política.

**Para uma campanha de herança divina e destino:**
Use cap00 + cap02 (Campo das Convergências). O subsistema de Herança Arcana (Dom Passivo + Marca Narrativa) é o mais modular e portável — funciona em qualquer campanha onde PCs têm origem especial.

**Para uma campanha institucional sombria:**
Use cap00 + cap03 (Ordem dos Glifeiros). O sistema de Glifos + Vínculo de Alma + Status por facção é a campanha mais mecanicamente densa do livro. Combine com cap04 para máxima intriga.

**Para a campanha inter-escolas completa:**
Use cap06 (Brambleshire, 8–12 sessões) + cap07 (Torneio, 3 sessões) como arco de intercâmbio. Funciona com qualquer paradigma de herança como base.

**Para opções de personagem sem campanha específica:**
Use cap05 (Manual do Estudante Arcano) para 11 arquétipos acadêmicos e o Apêndice para navegar todas as opções de kits e vantagens.

**Se não conhece nenhuma franquia:**
Use APENAS cap00 (o framework genérico). Ele foi escrito para funcionar sem qualquer material de referência. As franquias são instâncias — o framework é o jogo.
