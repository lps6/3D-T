# ÍNDICE DE SUPLEMENTOS — Academia Arcana Victory

> **Como usar este índice:** Os suplementos da Academia Arcana Victory foram consolidados em **8 Livros Temáticos** (L0–L7). Cada livro é um produto completo e autossuficiente — você não precisa de nenhum outro para jogar qualquer um deles. O livro base de 3D&T Victory é o único pré-requisito universal.
>
> Este índice detalha, para cada livro: os **capítulos**, as **influências** (franquias e suplementos-fonte consolidados), e as **mudanças relevantes** que o livro traz ao jogo — mecânicas novas, opções de personagem, escolas, grêmios, cidades e locais.
>
> Para os suplementos-fonte originais (F2.x–F4.x, S1–S6, C1–C3, T1–T4, Y1–Y3, K1, ALA, FAM1, N1) que foram consolidados nestes livros, ver `/Arquivo/`.
