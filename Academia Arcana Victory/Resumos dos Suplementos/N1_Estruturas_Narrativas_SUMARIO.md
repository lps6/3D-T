# Sumário: N1 - Estruturas Narrativas

Este documento é um guia para o Mestre sobre como usar estruturas narrativas de mídias populares (animes, livros, séries, videogames) para construir aventuras e arcos de campanha mais coesos e satisfatórios em Academia Arcana.

## Conceito Central
O suplemento argumenta que adaptar a *estrutura* de uma história não é plágio, mas sim "alfabetização narrativa". Ele ensina o Mestre a reconhecer, desmontar e aplicar esses padrões para suas próprias campanhas.

## Estruturas Apresentadas
O guia detalha oito estruturas principais, explicando seus estágios, como adaptá-las para a Academia, e os erros comuns a serem evitados:
- **De Anime:**
    1.  O Episódio de Anime (para sessões auto-contidas)
    2.  O Arco de Torneio (para competições como o MagiDerby)
    3.  O Arco de Festival Escolar (para drama social e emocional)
    4.  O Arco de Treinamento (para desenvolvimento de personagem focado)
- **De Livro:**
    5.  O Mistério Clássico (com mecânica de pistas e falsas soluções)
    6.  A Jornada do Herói Comprimida (para sessões de virada de personagem)
    7.  O Esquema de Sanderson (foco em foreshadowing e payoff de mecânicas)
- **De Outras Mídias:**
    8.  O Episódio de Série Live Action (estrutura de A-plot e B-plot)
    9.  O Arco de Videogame (exploração, boss intermediário, dungeon final)

## Mecânicas Principais
- **Fichas de Estrutura:** Para cada uma das 8 estruturas, há um template preenchível para o Mestre planejar suas sessões.
- **Índice Cruzado:** Uma tabela que recomenda quais estruturas funcionam melhor para cada um dos "Tons de Campanha" apresentados no Cap. 09.

## Conexão com o Cânone
Este é um guia de metanível para o Mestre, conectando-se a quase todos os outros suplementos ao fornecer as ferramentas para estruturar as aventuras neles propostas. É o "como fazer" para o "o que fazer" dos outros documentos.
