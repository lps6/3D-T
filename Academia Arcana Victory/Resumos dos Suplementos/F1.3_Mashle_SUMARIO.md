# Sumário: F1.3 - Mashle: Magic and Muscles

Este documento é uma análise da série *Mashle: Magic and Muscles* como fonte de inspiração para campanhas de Academia Arcana Victory.

## Conceito Central
Uma paródia que explora o que acontece quando um personagem sem nenhuma magia sobrevive e prospera em uma escola de elite mágica usando apenas força física absurda. A série questiona a validade de sistemas que medem o valor de uma pessoa por uma única métrica (aptidão mágica).

## Elementos Adaptáveis
- **O Caminho Sem Magia:** Um subsistema opcional para personagens com P(Poder) alto mas sem vantagens mágicas, permitindo-lhes usar Pontos de Esforço (PE) para realizar proezas físicas que replicam ou anulam a magia.
- **Subversão de Expectativas:** Oferece um modelo para aventuras onde a solução para um problema mágico não é mágica, forçando o grupo a pensar de forma criativa.
- **Hierarquia de Marcas Mágicas:** O sistema de "marcas" de Mashle é mapeado para a combinação de Habilidade + Mística em 3D&T, destacando como um personagem com P5/H1 seria um completo anômalo.
- **Fichas de Personagem:** Apresenta fichas prontas para Mash, Finn, Lance e Dot, adaptadas para os kits e grêmios da Academia Arcana.

## Conexão com o Cânone
Este fichamento oferece uma abordagem diferente para o tema do "não-mago", contrastando com A2 e A3. O "Caminho Sem Magia" pode ser combinado com o kit Lâmina Dançante para personagens como Li Meiling (F2.3).
