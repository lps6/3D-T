# Sumário: F2.3 - Cardcaptor Sakura

Este documento é uma análise da série *Cardcaptor Sakura* como fonte de inspiração para campanhas de Academia Arcana Victory.

## Conceito Central
A série popularizou o subgênero de "colecionadora mágica". A protagonista, Sakura, deve capturar e selar entidades mágicas (as Cartas Clow), cada uma com personalidade e poder únicos. O ato de capturar envolve mais empatia e compreensão do que força bruta.

## Elementos Adaptáveis
- **Aventuras Modulares:** O formato da série, onde cada "carta" é uma aventura de uma sessão, mapeia perfeitamente para o modelo de "Campanha de Exploração Segmentada" do Cap. 08.
- **Artefatos com Agência:** As Cartas Clow são tratadas como NPCs/entidades, não como itens. Isso pode ser usado para criar "Glifos Errantes" ou "Construtos sem mestre" que os PJs precisam entender para conter.
- **Poder pela Coleção:** A ideia de que o poder da protagonista cresce à medida que ela entende e cataloga mais entidades mágicas. Mecanicamente, sugere que a cada "X" artefatos selados, a personagem ganha um bônus permanente.
- **Fichas de Personagem:** Apresenta fichas prontas para Sakura, Tomoyo, Syaoran e Meiling, adaptadas aos kits e grêmios da Academia.

## Conexão com o Cânone
Este fichamento conecta-se ao sistema de Familiares (FAM1) para o guardião Kero-chan, ao suplemento A2 (Artefato-Dependente) pela ideia de poder ligado a um objeto, e ao Cap. 08 (Estrutura de Campanha).
